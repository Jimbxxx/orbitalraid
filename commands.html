Available Commands

  • /createwebhooks Create up to 20 webhooks and send raid messages to a server.
  • /raid Use all existing webhooks in a server to spam raid messages.
  • /wipe_webhooks Delete all webhooks from a server.
  • /checkguild Check if bot is in the server and fetch info.
  • /get_invite Generate an invite link to the specified server.

Commands will not work if the bot is not in the server, credits are still taken and will not be refunded if you have not added the bot.
